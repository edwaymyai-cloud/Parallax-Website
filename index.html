視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技術，讓網站背景的移動速度比前景慢。當訪客向下捲動頁面時，會產生 3D 效果，增添層次感並帶來更沉浸的瀏覽體驗。視差源自光學錯覺：人眼會把較近的物體看得更大，因而覺得遠處的物體移動得更慢。 這種錯覺早已被運用在不同媒介，營造逼真的效果。最早可以追溯到迪士尼《白雪公主與七個小矮人》等傳統動畫，以及《超級瑪利歐》等電玩遊戲。 隨著 CSS 與 HTML 的進步，視差效果逐漸發展成為今日網頁設計的一部分。

最早的瀏覽器視差滾動實作之一出現在 2007 年，由網頁開發者 Glutnix 在部落格分享範例程式碼與示範，使用 JavaScript 與 CSS2，支援當時的 Internet Explorer 6 等瀏覽器。不過直到 2011 年 HTML5 與 CSS3 普及後，這項技巧才在網頁設計師之間流行起來。支持者認為，視差是擁抱網頁流動性的簡單方式，可作為吸引用戶並提升網站體驗的工具。然而普渡大學 2013 年的研究指出：「……雖然視差滾動改善了使用者體驗的某些面向，但不一定能提升整體體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要理由：

  • 有助於說故事

    視差圖像非常適合用來說故事。好的設計重點在溝通——設計網站時，我們希望與訪客交流，講出關於產品、服務或品牌的精彩故事。視差滾動能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站，讓使用者不用跳轉也能讀完完整的品牌故事。

  • 有助提升使用者參與度

    視差圖像也能提升使用者參與度。降低網站跳出率是網頁設計的重要目標之一；跳出率指的是只瀏覽單一頁面就返回或關閉分頁的訪客比例。高跳出率會影響搜尋排名，因此任何能降低跳出率的作法都值得嘗試。精心打造的視差頁面能帶來動態體驗，讓使用者停留更久。


想在網站加入視差滾動效果時，可以參考以下建議：

  • 適度使用

    將視差效果運用在特定元素會比套用在所有頁面上更有效。可放在頁首、標題或僅在首頁使用，目的是強化網站設計，而非分散使用者注意力。

  • 圖像壓縮

    視差滾動會用到多種媒體檔案與 CSS 形狀來營造深度。請壓縮圖檔，避免過大的檔案拖慢載入速度，影響效能與體驗。

  • 善用色彩

    視差不僅關乎前景與背景圖像，色彩同樣重要。背景可使用較低飽和度的色調，前景則採用更鮮明的色彩，進一步強化深度感。
